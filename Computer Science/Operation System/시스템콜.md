시스템 콜
====================
- 운영체제가 제공하는 서비스를 사용자가 이용할 수 있도록 하는 것

#

### 커널모드, 사용자 모드
```
메모리 및 하드웨어 자원에 직접 액세스 가능/불가능으로 구분
```
- 커널모드 : 모든 시스템 메모리 접근 가능, 모든 CPU 명령 실행 가능
- 사용자 모드 : 하드웨어 직접 접근 불가, System call 호출 시 일시적으로 커널모드로 전환

#

### 사용자모드 -> 커널모드 : 시스템콜 사용
- 하드웨어의 접근이 필요할 경우 시스템 콜 사용하여 커널모드로 전환
- 끝나면 다시 사용자모드로 전환
